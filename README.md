# Install
Yarn install

Yarn dev

## Technology

PostgresQL, supabase, NextJS, TypeScript, React, and Tailwind.

Utilizes OpenAI to make movie recommendations and MovieDBAPI for dynamic movie data.

## Description

Free to use Movie website. Find movies using AI, find random movies the old fashioned way, Sign in though Google, add movies to your list, create groups, invite others to your group, share movies with friends in group lists, see trailers, get links to the url of the movie you want to watch.
